# cornavirus-tracker-cli

Track coronavirus from cli

## Screenshot

<img src="https://i.ibb.co/cxJkRHf/screenshot.png" width="960" height="720">

### Prerequisites

* node
* npm


### Installing

```
npm install coronavirus-tracker-cli -g
```

### Running

```
corona
```
