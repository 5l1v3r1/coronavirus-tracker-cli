# cornavirus-tracker-cli

Track coronavirus from cli

## Screenshots

<img src="./screenshot.jpg" width="720" height="450">

### Prerequisites

* node
* npm


### Installing

```
npm install coronavirus-tracker-cli -g
```

### Running

```
corona
```
